# NeuroDocAI

AI-powered document intelligence system. Upload PDFs/DOCX files, ask natural language questions, and get cited answers with summaries and insights.

## Tech Stack

- **Backend**: FastAPI, SQLAlchemy, PostgreSQL, ChromaDB
- **Frontend**: Next.js, TypeScript, Tailwind CSS, shadcn/ui
- **AI**: Ollama (nomic-embed-text), Tesseract OCR
- **Deployment**: Docker

## Status

Currently in **MVP** phase. See [docs/PROJECT_SPEC.md](docs/PROJECT_SPEC.md) for full specification.
