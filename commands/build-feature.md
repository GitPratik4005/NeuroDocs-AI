# /build-feature

## Purpose
Implement a feature using RPIT loop

---

## Steps

### 1. Research
- Use Explore Agent
- Identify relevant files

### 2. Plan
- Break into:
  - API
  - services
  - models
- Ask clarifying questions

### 3. Implement
- Follow modular structure

### 4. Test
- Add tests
- Validate integration

### 5. Document
- Update docs
- Trigger knowledge agent
